## 깃허브 특강 정리

- git init : 깃허브에서 관리하는 컴퓨터에서 .git 파일이 만들어진다.   ->  깃허브에 git init이 만들어진다.
- git clone 레포지토리 주소 (원격 저장소에서 가져오기)
- git 은 실제 도구 코드관리,파일관리,코드관리(버전 관리 시스템) -> github는 실제 저장소 ,GitHub는  git을 사용자가 편리하게 사용할 수 있는 웹 서비스
